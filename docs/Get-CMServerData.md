# Get-CMServerData

## SYNOPSIS
Pull server data from the CM Database.

## SYNTAX

```
Get-CMServerData [<CommonParameters>]
```

## DESCRIPTION
This function will return all of the current server records from the SCCM Database with several useful pieces of data.
